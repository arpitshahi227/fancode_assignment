# FanCode SDET Assignment

## Prerequisites

- Python 3.x installed
- Install dependencies using:
```bash
pip3 install -r requirements.txt

```

## Running the Project and Tests

1. **To run the project**:
   ```bash
   python3 src/main.py
   ```

2. **To run the tests**
3. ```bash
    python3 -m unittest discover tests
    ```
